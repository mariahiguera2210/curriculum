Detalle Proyecto Database connection with H2

Este proyecto es un ejemplo de como realizar una conección a base de datos con H2 usando Spring Boot con JPA. El proyecto crea una base de datos de cervezas

  1. Hay que ingresar al link
    2. http://localhost:8080/h2-console
  2. Ingresar la contraseña: Password
  3. Abrir postman e ingrsar la siguiente ruta http://localhost:8080/api/v1/beer/beers
  4. Se envia la petición como se ve en la imagen
  5. Luego en el H2 se verifica la creacion de la nueva cerveza en la base de datos
